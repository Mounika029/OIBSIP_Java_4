# Online Exam System

## 📌 Objective
To develop a Java-based Online Examination System that allows users to take a quiz, answer multiple-choice questions, and view their score.

## 🛠 Tools & Technologies
- Java
- OOP Concepts
- Arrays / Collections
- Scanner (for user input)
- (Optional) Swing for GUI

## 🚀 Features
- User login system
- Multiple-choice questions
- Timer-based exam (if implemented)
- Auto submission after completion
- Score calculation
- Display result at the end

## 🧩 Steps Performed
1. Designed question and answer structure
2. Implemented user login functionality
3. Displayed questions one by one
4. Captured user answers
5. Calculated score based on correct answers
6. Displayed final result

## 📊 Outcome
The Online Exam System works successfully, allowing users to take a test and receive their score instantly.

## ▶️ How to Run
1. Open the project in your IDE (VS Code / IntelliJ)
2. Compile all Java files
3. Run the main class
4. Login and start the exam
5. Answer the questions to get the result
